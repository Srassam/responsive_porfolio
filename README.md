## User Story

```
AS a potential employee, 
I WANT to show current and future employers the types of work I've done and  the progression of acquired skills,
SO THAT I can be part of a team that will make an impact in our world. 
```

## Acceptance Criteria

```
BY updating my current portfolio
WHEN I click on the portfolio
THEN I am able to view 2 group project and two projects that I am proud of.
WHEN clicking on the resume link
THEN I am able to view my current resume.
WHEN clicking on the LinkedIn link
THEN I am able to view my current LinkedIn portfolio.
